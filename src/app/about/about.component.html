About DBTWeb and the Founder C.E.O Tony Trapp

I am originally from a small town called Tujunga near Los Angeles, California San Francisco in late 1999 where I got a job as an Assistant Pastry Chef at Elroy's restaurant under Executive Chef Marc Valiani which was at south of Market Street at Folsom street and Beale street. I was very excited but I always wanted to do web design and web development / web application development which was my real passion.

I started DBT web productions in 2001 which was designbytony.com, but then I condensed it to DBTweb. I started with a host of technologies such as HTML(Hypertext Markup Language), Javascript, CSS(Cascading Style Sheets), Flash with Action Script, Adobe Dreamweaver, ASP (Active Server Pages), XML(Extensible Markup Language), VBScript(Visual Basic Script Edition), XHTML(Extensible HyperText Markup Language), XSLT(Extensible Stylesheet Language Transformations) and MS Access (low performance database). I started all of these things at the same time while working on designing web sites.

It was very hard at first, but I got better over the next five years. In 2003 I converted to ASP.net framework 1.1 and converted to SQL(Sequel) server Database, T-SQL(Transact-SQL) and Jquery / Ajax(Asynchronous JavaScript and XML) and JSON (Javascript Object Notation).

I tried Coldfusion and PHP(Personal Home Page), but ASP.NET Web Forms is my preferred language and has everything built in that I need. I started on my own server which was Windows Server 2003, Flash Media Server, Kerio Mail Server and FTP server named Filezilla and I learned all about servers as well as serving up websites at the same time. I had 100 clients that I host with out advertising, in 2015 I converted from Flash Media Server to Wowza Server.

I was also interested in video and 3D and I got Cinema 4D, Adobe After Effects, Adobe Audition, Adobe fireworks and a lot of plugins for these programs. I started to learn how to make videos and 3D objects and combine them together. I was also interested in audio and sound effects so I started learning that as well. You can see the videos in my portfolio section.

Now I am doing Wordpress(content management system or CMS) and plesk(server administraton on the web), I started that in 2015, Windows Server 2019, SQL server 2019, .Net 8.0, MVC and Angular with Visual Studio 2022 / Visual Studio Code.. One more thing I reside in Staten Island New York.
